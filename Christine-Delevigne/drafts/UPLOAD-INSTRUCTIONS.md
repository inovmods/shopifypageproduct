## 🚀 **Upload Steps:**

1. **Download the fixed ZIP:**
   - Use `Blueprint-Theme-Fixed.zip` (NOT the old Blueprint-Theme.zip)

2. **Upload to Shopify:**
   - Go to Shopify Admin → Online Store → Themes
   - Click "Upload theme"
   - Select `Blueprint-Theme-Fixed.zip`
   - Click "Upload"

3. **Should work now!** ✅
   - The error "missing template layout/theme.liquid" should be resolved
   - All required Shopify theme files are now included

## 🎨 **After Upload:**

1. **Activate the theme:**
   - Click "Actions" → "Publish" on the Blueprint theme

2. **Customize the theme:**
   - Click "Customize" to open the theme editor
   - Configure colors, fonts, and layout in Theme Settings
   - Add your logo and favicon

3. **Set up 3D model:**
   - Create your 3D t-shirt model in Spline (spline.design)
   - Copy the embed URL
   - In theme customizer → Hero 3D section → paste Spline URL
   - Enable "Enable 3D model" option
