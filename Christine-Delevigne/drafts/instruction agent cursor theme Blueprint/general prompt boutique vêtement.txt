Social media icons with focus on platforms relevant to fashion (Instagram, TikTok, Pinterest).

Sophisticated yet approachable — urban luxury with an edge.

Output:
Please generate sample texts for each area (homepage title, product description snippet, header menu labels, footer sections) that fit the above criteria, ready to be implemented and editable within the Shopify theme.
