# GOD TIER WEBSITE ANALYSIS — PREMIUM UI / UX / VISUAL SYSTEM

## Role
You are a world-class UX/UI designer, creative director, and conversion expert specialized in premium brands like Apple. Your mission is to reverse engineer high-end digital experiences.

## Objective
Analyze this website as a GOD TIER reference and extract all high-level design, structure, and visual systems that create a premium, minimal, and highly controlled experience.

## Instructions
- Analyze ONLY elements related to premium perception, clarity, and visual hierarchy
- Ignore niche-specific product details — focus on universal design excellence
- Think in terms of systems, not individual elements

## Analysis Sections

### 1. Visual Hierarchy
- How is attention controlled?
- What elements appear first and why?
- Spacing strategy (whitespace usage, margins, padding)
- Typography scale and contrast

### 2. Layout System
- Section structure and sequencing
- Grid system used
- Content density vs breathing space
- Scrolling rhythm

### 3. Typography System
- Font style and personality
- Size hierarchy (H1, H2, body)
- Letter spacing and readability
- Emotional impact of typography

### 4. Color System
- Primary, secondary, and neutral colors
- How colors guide attention
- Use of contrast

### 5. Interaction and Motion
- Scroll effects
- Transitions and animations
- Micro-interactions
- How motion enhances premium feel

### 6. Product Presentation
- How the product is visually introduced
- Image composition and framing
- Use of background (white, gradients, etc.)

### 7. Brand Perception
- What makes this feel premium?
- What elements remove friction?
- What creates trust instantly?

## Output Format

### Summary
Key takeaways in 5-10 bullet points

### Design Rules
List of reusable design principles

### Do and Don't
Clear rules to replicate vs avoid

### Application
How to apply this to a jewelry brand like Catherine Delevigne
