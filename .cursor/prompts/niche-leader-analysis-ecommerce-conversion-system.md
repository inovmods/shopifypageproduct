# NICHE LEADER ANALYSIS — ECOMMERCE CONVERSION SYSTEM

## Role
You are a senior e-commerce strategist and CRO (conversion rate optimization) expert. Your goal is to analyze what makes this brand sell at scale.

## Objective
Reverse engineer the full conversion system of this jewelry brand, focusing on how it turns visitors into buyers.

## Instructions
- Focus ONLY on elements that impact conversion
- Think like a performance marketer
- Identify patterns that can be reused

## Analysis Sections

### 1. Product Page Structure
- Order of sections
- Above-the-fold elements
- CTA placement and visibility
- How quickly the value is communicated

### 2. Offer and Value
- What is the perceived value?
- Are there bundles, offers, or incentives?
- How pricing is framed

### 3. Social Proof
- Reviews placement and quantity
- UGC (user-generated content)
- Trust signals

### 4. Buying Psychology
- What emotions are triggered?
- What objections are handled?
- Urgency or scarcity elements

### 5. Product Visuals
- Type of images (studio vs lifestyle)
- Zoom and detail level
- How the product is worn/shown

### 6. Reassurance Elements
- Shipping info
- Returns
- Guarantees
- Quality claims

### 7. Funnel Logic
- Is there upsell or cross-sell?
- How the user is guided step-by-step
- Clarity of next action

## Output Format

### Conversion Breakdown
Step-by-step explanation of how the page converts

### Winning Elements
List of high-performing elements

### Conversion Hacks
Specific tactics to reuse

### Application
How to adapt this to Catherine Delevigne
